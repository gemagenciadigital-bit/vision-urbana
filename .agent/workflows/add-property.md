---
description: Steps to integrate a new property into the Vision Urbana website
---

Follow these steps whenever adding a new property:

1. **Understand Property Details**: Review the assets and information provided by the user (images, location, price, specs).
2. **Implementation (Local)**: 
   - Create a new HTML file in `web/detalles/[property-name].html`.
   - Update `data/propiedades.json` with the new property object.
   - Verify all local paths and links.
3. **User Review**: **MANDATORY**. Notify the user that the files have been created/updated locally. Wait for their confirmation that everything looks correct.
